# Dagron

Dagron ([sic](http://www.urbandictionary.com/define.php?term=DAGRON)) is a
Sinatra-based app for displaying TMX maps for tabletop games.

## Installation

Add this line to your application's Gemfile:

    gem 'dagron'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dagron
